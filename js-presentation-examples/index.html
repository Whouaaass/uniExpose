OnFocus

Este ejemplo muestra como usar el evento onfocus

El evento onfocus se dispara cuando el elemento recibe el foco, es decir es seleccionado por el usuario

Este evento se diferencia de onclick porque el elemento puede recibir el foco por varias razones, por ejemplo, mediante el uso de TAB

OnReset

Este ejemplo muestra como usar el evento onreset

Para usar onreset, se debe de usar en un <form /> que permita el control de varios elementos tipo input y al cual se le hará el reseteo

En este caso se usa el evento onreset para mostrar un mensaje en la pantalla cuando se hace el reseteo del formulario

window.LocalStorage

la función de localStorage es una forma de almacenar información en el navegador, es decir en el cliente, en este caso en el navegador de escritorio

Para usar localStorage, se debe de crear una variable de tipo localStorage y asignar un valor a ella

localStorage.setItem('nombreVariable', 'valor');

Para leer el valor de una variable, se debe de usar el método getItem

localStorage.getItem('nombreVariable');
